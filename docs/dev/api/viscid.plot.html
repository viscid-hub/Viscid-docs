Module contents¶

Package of modules that make it convinient to plot fields using other libraries (Matplotlib / Mayavi)

Note

vpyplot and vlab are not imported by default since these import pylab and mlab, which has side-effects. To use them, you must use the imports explicitly,

>>> from viscid.plot import vpyplot as vlt
>>> from viscid.plot import vlab
